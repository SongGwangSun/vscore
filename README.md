# 스포츠 점수판 앱

탁구와 배드민턴을 위한 스마트폰/태블릿 점수판 웹 앱입니다.

## 주요 기능

### 🎮 게임 선택
- 피클볼 (🎾)
- 탁구 (🏓)
- 배드민턴 (🏸)
- 족구 (🏐)
  
- **모바일 전체화면 게임 선택 화면**

### ⚙️ 게임 설정
- 승리 점수: 11점 15점 21점
- 세트 수: 3세트 또는 5세트
- **모바일 친화적 설정 인터페이스**

### 📊 점수 관리
- **한 번 터치**: 점수 증가 (+1)
- **두 번 연속 터치**: 점수 감소 (-1)  -- 삭제
- **음성 안내**: 점수 변경 시 음성으로 점수 불러주기
- 10 을 열로 발음 -> 십으로 변경경
- **실수 수정**: 마지막 점수 취소 기능
- **모바일 최적화 터치**: 정확한 터치 감지 및 반응

### 🏆 세트 관리
- 상단에 세트 점수 표시
- 세트 승리 시 자동 다음 세트 진행
- 게임 종료 시 승자 발표
- **모바일에서 큰 점수 표시**
- 각각의 게임 룰에 따른 서브 교체
- 서브권 점수 빨간색으로 구분

### 📱 반응형 디자인
- 스마트폰과 태블릿 모두 지원
- 세로/가로 모드 자동 최적화
- 터치 친화적 인터페이스
- **iOS/Android 최적화**
- **iPhone X 이상 안전 영역 지원**
- **다크 모드 지원**

## 모바일 최적화 기능

### 🎯 터치 최적화
- **정확한 터치 감지**: 터치 시간과 거리 기반 필터링
- **더블 탭 방지**: 실수로 인한 줌 방지
- **터치 피드백**: 시각적 터치 반응
- **접근성 지원**: 키보드 네비게이션

### 📱 화면 최적화
- **전체화면 게임 선택**: 모바일에서 전체 화면으로 게임 선택
- **큰 점수 표시**: 모바일에서 점수를 크게 표시
- **명확한 터치 영역**: 점수 터치 영역을 절반씩 크게 구분
- **세트 점수 표시**: 상단에 세트 점수 명확히 표시

### 🔧 기술적 최적화
- **iOS Safari 최적화**: -webkit- 접두사 및 터치 이벤트 최적화
- **Android Chrome 최적화**: touch-action 속성 활용
- **PWA 지원**: 앱 설치 및 오프라인 사용 가능
- **성능 최적화**: 부드러운 애니메이션 및 반응성

## 사용 방법

### 1. 게임 시작
1. 피클볼 탁구 또는 배드민턴 또는 족구 선택 (전체화면 선택 화면)
2. 승리 점수와 세트 수 설정
3. "게임 시작" 버튼 클릭

### 2. 점수 입력
- **플레이어 1 점수**: 왼쪽 절반 영역 터치
- **플레이어 2 점수**: 오른쪽 절반 영역 터치
- **점수 감소**: 빠르게 두 번 연속 터치  -- 삭제
- **모바일 최적화**: 큰 터치 영역과 명확한 시각적 구분

### 3. 게임 제어
- **세트 리셋**: 현재 세트 점수 초기화
- **실수 수정**: 마지막 점수 취소
- **새 게임**: 처음부터 다시 시작

### 4. 키보드 단축키 (데스크톱)
- **1**: 플레이어 1 점수 증가
- **2**: 플레이어 2 점수 증가
- **R**: 세트 리셋
- **Z**: 마지막 점수 취소

### 5. 모바일 특별 기능
- **화면 방향 고정**: 게임 중 세로 모드 유지
- **전체화면 모드**: 게임 점수판 전체화면 표시
- **터치 최적화**: 정확한 터치 감지 및 반응

## 기술 스택

- **HTML5**: 시맨틱 마크업, 접근성 지원
- **CSS3**: 반응형 디자인, Flexbox, Grid, 모바일 최적화
- **JavaScript**: 게임 로직, 터치 이벤트, 음성 합성
- **PWA**: 오프라인 지원, 앱 설치 가능
- **모바일 최적화**: iOS/Android 특화 기능

## 설치 및 실행

### 웹에서 사용
1. 브라우저에서 `index.html` 파일 열기
2. 또는 웹 서버에 업로드하여 사용

### PWA 앱으로 설치
1. Chrome/Edge에서 앱 열기
2. 주소창 옆 "설치" 버튼 클릭
3. 홈 화면에 앱 아이콘 생성

### 모바일에서 최적 사용법
1. **Chrome/Edge에서 열기**
2. **PWA로 설치** (권장)
3. **세로 모드에서 사용**
4. **전체화면으로 실행**
